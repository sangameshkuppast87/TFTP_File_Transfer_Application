3. Start date:     16/02/2026

4. End date:       20/02/2026

5. Description:	   → The TFTP Server is a UDP-based file transfer service implemented in C that listens for client requests and serves files upon demand.
		   → It supports handling Read Requests (RRQ) and Write Requests (WRQ) from multiple clients sequentially.
		   → The server manages file I/O operations, packet segmentation into data blocks, and acknowledgement (ACK) generation for reliable transfer.
		   → It implements error handling for invalid requests, missing files, access violations, and transfer timeouts.
		   → The project demonstrates core networking and operating system concepts such as socket communication, file system interaction, process control, and protocol implementation.
