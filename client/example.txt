5. Description:  → The TFTP Client is a UDP-based file transfer application developed in C that allows users to upload and download files from a remote TFTP server.
		 → It supports sending Read Requests (RRQ) and Write Requests (WRQ) according to the TFTP protocol specification.
		 i→ The client handles data transmission in fixed-size blocks, acknowledgement (ACK) processing, and retransmission on timeout.
		 → It supports both octet (binary) and netascii (text) transfer modes.
		 → The project demonstrates networking concepts such as socket programming using UDP, packet formatting, timeout handling, and reliable data transfer over an unreliable protocol.
